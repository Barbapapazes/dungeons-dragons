# Démarrer le jeu

Avant même de commencer à joueur au jeu, il va falloir le démarrer. Pour cela, il vous faut à minima python 3.9 (opens new window) installé sur votre machine avec pip.

Ensuite, il faut démarrer un terminal dans le path du projet puis lancer les commandes suivantes :

# install a local environment
$ pip install pipenv
# install all packages
$ pipenv install
# start the project
$ pipenv run start

Voilà ! Vous êtes fin prêt à commencer à découvrir le jeu ! Mais vous pouvez aussi continuer à lire le guide !
